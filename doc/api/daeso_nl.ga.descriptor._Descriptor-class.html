Abstract class for feature descriptions

A _Descriptor instance describes the features of a dataset (instances). The order is significant. It wraps an ordered dictionary where the keys are unique feature names and the values are Feat instances. Each Feat instance defines the properties of a feature (it's type, the associated feature function, etc.).
