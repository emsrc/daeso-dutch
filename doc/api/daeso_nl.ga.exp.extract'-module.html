Module extract'

source code

Extract features, create instances and true pgc files

This step extracts features from the parallel graph corpora and creates Timbl instances. Each instance represents a possible alignment & labeling from a node n in the source graph to a node m in the target graph. In addition, true pgc files are generated, which may differ from the original parts because of ignored nodes and/or graphs.

Extract features from source corpus files, produce instance files and true corpus files.

Parameters:
  • extractor - Extractor instance for feature extraction from graph pairs
  • selector - boolean graph pair selection function
  • part_fns - list of corpus filenames
  • inst_fns - list of instance filenames to be created
  • true_fns - list of true corpus filenames to be created
  • binary - save corpus instances in binary rather than text format

    Note that the true corpus files may be substantially different from the original corpus files because of node and graph selection.
