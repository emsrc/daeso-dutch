Classify instances using Timbl

Parameters:
  • train_inst_fns - a list of instance filenames for training
  • test_inst_fns - a list of instance filenames for testing
  • out_fns - list of classifier output files to be created
  • log_fn - classifier log file to be created; ignored if keyword log is false
  • clas_dir - directory for creating classifier output files; ignored if out_fns is given
  • descriptor - a Descriptor instance, required to infer the feature metrics for Timbl, unless a TimblFile is supplied; ignored if timbl is supplied
  • timbl - a tailored TimblFile instance; notice that it must at least set the verbosity options +vo, +vdb, +vdi, and the -m option to specify that the administrative features must be ignored.
  • options - list of additional Timbl options, excluding -f, -m, +vo, +vdb, +vdi
  • log - log Timbl's standard output and error streams to file
Returns:
list of Timbl output filenames

classify_file_cv(inst_fns, test_inst_fns=None, out_fns=None, log_fns=None, clas_dir=None, descriptor=None, timbl=None, options='', n=None, log=False)

source code 

Classify instance using Timbl in a cross-validation procedure.

Parameters:
  • inst_fns - a list of instance filenames for training; if no test_inst_fns is supplied, the same files will be used for testing, otherwise they are used for training only
  • test_inst_fns - a list of instance filenames for testing; this allows for down-sampling of the training instances without affecting the test instances
  • out_fns - list of classifier output files to be created
  • log_fns - list of classifier log files to be created; ignored if keyword log is false
  • clas_dir - directory for creating classifier output files; ignored if out_fns is given
  • descriptor - a Descriptor instance, required to infer the feature metrics for Timbl, unless a TimblFile is supplied; ignored if timbl is supplied
  • timbl - a tailored TimblFile instance; notice that it must at least set the verbosity options +vo, +vdb, +vdi, and the -m option to specify that the administrative features must be ignored.
  • options - list of additional Timbl options, excluding -f, -m, +vo, +vdb, +vdi
  • n - limit on the number of cross-validations performed (by default equals the number of instance filenames)
  • log - log Timbl's standard output and error streams to file
Returns:
list of Timbl output filenames
