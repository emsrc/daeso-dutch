multi_semantic_sim(self, from_spec_list, to_spec_list)

source code 

Calculate semantic similarity scores on four of Daeso similarity relations - restates, specifies, generalizes and intersects - for all possible combinations of lexical units from the two lists. Return a matrix with len(from_spec_list) rows and len(to_spec_list) columns which stores the similarity measures, i.e. tuples of scores on each of the four semantic relations. Based on Lin's similarity measure.
