pp_term_align(nodes, graphs, alignment, **kwargs)

source code 

A node preprocessing function that computes the number of aligned terminals for a given pair of source and target nodes.

Assumes the "_yield" attribute on nodes as computed by the pp_yield grap preprocessing functions.

Provides the node attributes: _inside: terminals aligned to terminals inside the other node _outside: aligned outside the other node or aligned to non-terminals _none: unaligned terminals
