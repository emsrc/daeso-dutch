pp_cornet_sim(graphs)

source code 

A preprocessing function

Requires access to the Cornetto database. By default we assume a DaesoCornet server is running on localhost. Otherwise you have to create your own client by calling the function "create_cornet_server_proxy". Alternatively, you can load Cornetto directly by calling the function "load_cornet".
