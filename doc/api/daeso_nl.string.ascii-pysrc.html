  6  Support for encoding in ascii where iso-88591-1 characters (i.e. those code 
  7  points which can be encoded in iso-8859-1 aka latin-1) beyond ascii chars 
  8  (i.e. those code points which can be encoded in ascii) are replaced with the 
  9  closest ascii equivalant. For example, 'e-accent-grave' is replaced by plain 
 10  'e', 'o-umlaut' is replaced by plain 'o', etc.
