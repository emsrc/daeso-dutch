A thin Python wrapper around the Alpino parser for Dutch. The input must be a tokenized Dutch sentence in unicode. The output is a parse tree in xml format in unicode. Relies on Pexpect for running Alpino as a subprocess and providing non-blocking communication through a pipe. Requires a local copy of the Alpino parser binary and an appropriate setting of the ALPINO_HOME environment variable.
